Project Nexus Manifest

Build a hybrid Local/Cloud "Personal Intelligence Command Center" (PICC) that automates RF engineering research and code development. This project serves as a living curriculum for the user to master LLMs, Agentic Workflows, and the Model Context Protocol (MCP).

Tech Stack

  • Interface: Gemini CLI / OpenCode
  • Backend: LM Studio (Local Qwen 35B)
  • Memory: ChromaDB
  • Standard: MCP (JSON-RPC)

Defensive Pillars

  • JSON Hygiene & Defensive Parsing
  • System Directives & Anchoring
  • VRAM Management (8GB/12GB Hard Limits)

🛠️ AI Troubleshooting Guide

Trap 1: The Empty Response

Cause: Context Overflow or Model Refusal. Fix: Reduce tokens or lower temperature.

Trap 2: The Template Collision

Cause: Breaking User-Assistant alternation. Fix: Ensure strict role-playing in loops.

Trap 5: The Latency Timeout

Cause: Local GPU TTFT exceeds 60s. Fix: Set client timeout to 300.0s.

Phase 1: The Foundation

Subject: Inference, Tokens, and the Physical Layer of AI

Master the mechanics of how data moves from your hardware to the model and back.

🧪 Lab Reference: LABS/benchmark_inference.py

Run this to measure your Tokens Per Second (t/s). Expect ~19 t/s on an RTX 3080 Ti.

Key Terminology

  • Token: Atomic unit of data. (IQ Sample)
  • Context Window: Observation buffer. (FFT Window)

Success Criteria

Benchmarked local inference > 10 t/s.

Phase 3: The Hand (Agency)

Subject: Function Calling and System Directives

Give the AI "Hands" to touch your terminal and filesystem.

🧪 Lab Reference: LABS/agent_multitool.py

Build an agent that creates a script and executes it autonomously.

Key Terminology

  • System Directive: High-priority behavior rules.
  • Anchoring: Preventing context drift.

Success Criteria

Agent completed a closed-loop Create-Run-Verify task.
