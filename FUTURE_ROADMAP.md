# Project Nexus: Future Roadmap
## Advanced Mastery (Phases 9-13)

### Phase 9: The Eye (Vision-Language Models)
- **Objective:** Master VLMs to analyze RF diagrams and GNU Radio flowgraphs.
- **Key Tech:** Llama-3.2-Vision, Moondream2.

### Phase 10: The Evolver (Self-Building Tools)
- **Objective:** Build an agent that can write its own MCP tools and self-upgrade.
- **Key Tech:** Meta-programming, Recursive Loops.

### Phase 11: The Analyst (IQ Intelligence)
- **Objective:** Bridge raw signal processing with AI-driven classification.
- **Key Tech:** NumPy FFTs, RF-specific Transformers.

### Phase 12: The Scout (Real-Time Web Intelligence)
- **Objective:** Bypass training cutoffs with real-time web search agents.
- **Key Tech:** Tavily API, Perplexity Integration.

### Phase 13: The Dashboard (Agentic UI)
- **Objective:** Move from CLI to a visual "Command Center" dashboard.
- **Key Tech:** Streamlit, WebSockets, React.
